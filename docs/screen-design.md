# 画面設計 — kaji-note

## 設計方針

- **画面数を最小限に**: 4 画面 + 1 モーダルで全機能を網羅する
- **ノートの世界観**: 罫線・手書き風フォント・クリーム紙色を基調とし、実際のノートに書き込まれた手順書を連想させる UI とする
- **モバイルファースト**: 基準画面幅 375px、片手操作を前提とした要素配置

│                         │

│                         │
│  ┌─────────────────┐    │

│  └─────────────────┘    │

## S-02 ノート棚（ホーム）

**目的**: 手順書一覧の表示・検索・新規作成への導線

```
┌─────────────────────────┐
│  kaji-note       ⚙      │  ← ヘッダー（グループ設定へ）
│ ─────────────────────── │
│  🔍 手順を検索...        │  ← 検索バー
│ ─────────────────────── │
│                         │
│  ┌────────┐ ┌────────┐  │
│  │ 風呂掃除│ │ 洗濯  │  │  ← ノートの背表紙風カード
│  │ 🕐 明日 │ │ ✅ 済  │  │    （次回予定日 or 実施済バッジ）
│  └────────┘ └────────┘  │
│                         │
│  ┌────────┐ ┌────────┐  │
│  │料理下準備│ │ゴミ出し│  │
│  │ 🔴 期限超│ │ 🕐 今週 │  │
│  └────────┘ └────────┘  │
│                         │
│           ╋             │  ← 新規作成ボタン（右下固定）
└─────────────────────────┘
```

**要素:**
- ヘッダー: アプリ名 / グループ設定ボタン（⚙）
- 検索バー（キーワード・カテゴリ絞り込み）
- ノートカード一覧（グリッド表示）
  - タイトル
  - 次回実施予定日 / 実施済バッジ / 期限超過バッジ
- 新規作成ボタン（╋、右下固定）

**カードバッジの種類:**

| バッジ | 意味 |
|--------|------|
| 🔴 期限超過 | 実施予定日を過ぎている |
| 🕐 〇〇日 | 次回実施予定日を表示 |
| ✅ 実施済 | 当日実施済み |
| （なし） | 頻度未設定 |

## S-03 手順ページ（閲覧）

**目的**: 手順書の詳細閲覧・実行モードへの切り替え

```
┌─────────────────────────┐
│  ←          ✏️          │  ← ヘッダー（戻る / 編集）
│                         │
│  風呂掃除               │  ← タイトル（手書き風フォント・大）
│  週1回 | 次回: 5/3(土)  │  ← 頻度・次回予定日
│ ─────────────────────── │  ← 罫線
│                         │
│  1. 換気扇をまわす      │
│                         │
│  2. 洗剤をスプレーする  │
│    ┌─────────────────┐  │
│    │      写真        │  │  ← 貼り付け写真風（枠・影あり）
│    └─────────────────┘  │
│                         │
│  3. ブラシでこする      │
│                         │
│  ・・・                 │
│ ─────────────────────── │
│  ┌─────────────────────┐│
│  │    ▶ 実行モード    ││  ← 実行開始ボタン（固定）
│  └─────────────────────┘│
└─────────────────────────┘
```

**実行モード（画面内切り替え）:**

```
┌─────────────────────────┐
│  ✕         2 / 5        │  ← 閉じる / ステップ進捗
│ ─────────────────────── │
│                         │
│  2. 洗剤をスプレーする  │  ← 現在ステップ（大きく表示）
│                         │
│  ┌─────────────────────┐│
│  │         写真         ││
│  └─────────────────────┘│
│                         │
│                         │
│  ┌────────┐ ┌─────────┐ │
│  │  ← 戻る│ │ 次へ → │ │  ← ステップ移動
│  └────────┘ └─────────┘ │
│                         │
│  最終ステップで「完了」  │  ← 完了時に実施日を記録
└─────────────────────────┘
```

**要素（閲覧時）:**
- ヘッダー: 戻るボタン / 編集ボタン（✏️）
- タイトル・頻度・次回予定日
- ステップ一覧（番号 + テキスト + 写真）縦スクロール
- 実行モード開始ボタン（固定フッター）

**要素（実行モード時）:**
- 閉じるボタン / ステップ進捗インジケーター
- 現在ステップのテキスト・写真（大きく表示）
- 前へ / 次へボタン
- 最終ステップで「完了・実施済みにする」ボタン

## S-04 書き込み（作成・編集）

**目的**: 手順書の新規作成および既存手順書の編集

```
┌─────────────────────────┐
│  ✕              保存    │  ← ヘッダー（キャンセル / 保存）
│ ─────────────────────── │  ← 罫線
│                         │
│  タイトルを書く_________ │  ← タイトル入力（罫線上）
│                         │
│  頻度: [設定なし ▾]     │  ← 頻度ピッカー
│                         │
│ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ │  ← 破線（ステップ区切り）
│  1. ________________    │
│     ________________    │  ← テキスト入力（罫線上）
│                         │
│     [📷]  [🎤]  [✨AI] │  ← 写真・音声・AI入力ボタン
│    ┌─────────────────┐  │
│    │   添付写真プレビュー│ │  ← 添付時に表示
│    └─────────────────┘  │
│                         │
│ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ │
│  2. ________________    │
│     ________________    │
│     [📷]  [🎤]  [✨AI] │
│                         │
│  ＋ ステップを追加      │  ← ステップ追加ボタン
│                         │
└─────────────────────────┘
```

**要素:**
- ヘッダー: キャンセル（✕）/ 保存ボタン
- タイトル入力欄
- 頻度設定ピッカー（設定なし / 毎日 / 週次 / 月次 / 季節ごと / カスタム）
- ステップ（繰り返し）
  - テキスト入力欄（罫線上に直接書く感覚）
  - 写真追加ボタン（📷 カメラ / ライブラリ）
  - 音声入力ボタン（🎤）
  - AI 生成ボタン（✨）
  - 添付写真プレビュー（削除可）
- ステップ追加ボタン
- ステップ長押しで並べ替え（ドラッグ）・削除メニュー

**AI 機能の呼び出しタイミング:**

| ボタン | 動作 |
|--------|------|
| タイトル入力後に「✨ AIで生成」 | タイトルを元にステップ一覧を自動生成（UC-07-1） |
| ステップ内「✨」 | 既存ステップを文脈に次ステップを提案（UC-07-2） |
| ステップ内「📷」で写真添付後 | 写真から説明文を自動生成（UC-07-3） |
| ステップ内「🎤」 | 音声入力 → テキスト変換 → AI 整形（UC-07-4） |

## S-05 グループ設定（モーダル）

**目的**: グループ管理・メンバー招待・プロフィール設定・手順書の共有設定

```
┌─────────────────────────┐
│           ✕             │  ← 閉じる
│  ─────────────────────  │
│                         │
│  👤 山田 花子           │  ← プロフィール（タップで編集）
│                         │
│  ─────────────────────  │
│  グループ: 山田家        │
│                         │
│  メンバー               │
│   ・山田 花子（管理者）  │
│   ・山田 太郎            │
│   ・山田 次郎            │
│                         │
│  ┌─────────────────┐    │
│  │ ＋ メンバーを招待 │   │
│  └─────────────────┘    │
│                         │
│  ─────────────────────  │
│  共有リンク             │
│  [リンクをコピー]        │
│                         │
│  ─────────────────────  │
│  ログアウト             │
│                         │
└─────────────────────────┘
```

**要素:**
- プロフィール（アイコン・表示名、タップで編集）
- グループ名・メンバー一覧（役割表示）
- メンバー招待ボタン（リンク or メールアドレス）
- 閲覧専用共有リンクのコピー
- ログアウト

## デザイントークン

| 項目 | 値 | 意図 |
|------|----|------|
| 背景色 | `#FAF6EE`（クリーム） | ノートの紙 |
| 罫線色 | `#C9C0A8` | ノートの罫線 |
| アクセントカラー | `#2A5FA5`（インクブルー） | ボールペンのインク |
| 危険アクション | `#C0392B`（赤） | 赤ペンの書き込み |
| タイトルフォント | 手書き風（Caveat / Klee One） | ノートのタイトル見出し |
| 本文フォント | ゴシック系（Noto Sans JP） | 可読性重視 |
| カード影 | `box-shadow: 2px 3px 8px rgba(0,0,0,0.12)` | 貼り付けた紙の浮き |
| 写真枠 | 白背景・1px ボーダー・2° 傾き | ノートに貼った写真 |
